# DomeneTech - Soluções em TI

Site institucional e portfólio da **DomeneTech**, desenvolvido para apresentar serviços, projetos e formas de contato profissional.

## 🚀 Sobre o projeto

A DomeneTech é uma iniciativa voltada para o desenvolvimento de soluções digitais, incluindo sites, sistemas web, automações e suporte em TI.

O site foi criado com foco em apresentar os serviços de forma clara, moderna e responsiva.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Responsividade

##  Funcionalidades

- Página inicial institucional
- Seção sobre
- Apresentação de serviços
- Portfólio de projetos
- Links para contato
- Layout responsivo

##  Projetos apresentados

- RoletaSorteio
- Gerenciador de Tarefas
- Landing Page Barbearia Prime
